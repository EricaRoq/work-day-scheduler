# Work Day Scheduler

This is a simple calendar application that allows a user to save events for each hour of the day by modifying starter code. This app will run in the browser and will show correct date and time on the top.

# How it works

Simply click on the day that you wish to schedule and write your comment. Then, click save to store the information.
